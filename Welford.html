Welford's Recursive Variance Formula

Tue 07 March 2017

Filed under maths

Tags calculus

We detail the calculation of the recursive formula:

\begin{equation*} S_n = \sum_{i=1}^n (x_i - \bar{x}_n)^2 \end{equation*}

We introduce \(\bar{x}_{n-1}\) to relate \(S_n\) with \(S_{n-1}\):

\begin{equation*} S_n = \sum_{i=1}^n ((x_i - \bar{x}_{n-1}) + (\bar{x}_{n-1} - \bar{x}_n))^2 \end{equation*}

We expand the expression:

\begin{equation*} S_n = \sum_{i=1}^n (x_i - \bar{x}_{n-1})^2 + n(\bar{x}_{n-1} - \bar{x}_n)^2 - 2 (\bar{x}_{n-1} - \bar{x}_n) \sum_{i=1}^n (x_i - \bar{x}_{n-1}) \end{equation*}
\begin{equation*} S_n = \sum_{i=1}^n (x_i - \bar{x}_{n-1})^2 + n(\bar{x}_{n-1} - \bar{x}_n)^2 - 2 n (\bar{x}_{n-1} - \bar{x}_n) (\bar{x}_n - \bar{x}_{n-1}) \end{equation*}
\begin{equation*} S_n = \sum_{i=1}^n (x_i - \bar{x}_{n-1})^2 - n(\bar{x}_{n-1} - \bar{x}_n)^2 \end{equation*}
\begin{equation*} S_n = S_{n-1} + (x_n - \bar{x}_{n-1})^2 - n (\bar{x}_{n-1} - \bar{x}_n)^2 \end{equation*}

Using the relation between the mean \(\bar{x}_n\) and \(\bar{x}_{n-1}\)

\begin{equation*} n \bar{x}_n = (n-1) \bar{x}_{n-1} + x_n \end{equation*}

Then observing the following equality:

\begin{equation*} x_n - \bar{x}_{n-1} = n (\bar{x}_n - \bar{x}_{n-1}) \end{equation*}

Using the first equality, we can reduce the last two terms in \(S_n\)

\begin{equation*} S_n = S_{n-1} + n(n-1) (\bar{x}_n - \bar{x}_{n-1})^2 \end{equation*}

Then by reordering a bit:

\begin{equation*} S_n = S_{n-1} + n (\bar{x}_n - \bar{x}_{n-1}) \times (n-1) (\bar{x}_n - \bar{x}_{n-1}) \end{equation*}

We recognize:

\begin{equation*} S_n = S_{n-1} + (x_n - \bar{x}_{n-1}) \times (n-1) (\bar{x}_n - \bar{x}_{n-1}) \end{equation*}
\begin{equation*} S_n = S_{n-1} + (x_n - \bar{x}_{n-1}) \times \frac{n-1}{n} (x_n - \bar{x}_{n-1}) \end{equation*}

But we know

\begin{equation*} \bar{x}_{n-1} = \frac{n \bar{x}_n - x_n}{n-1} \end{equation*}
\begin{equation*} x_n - \bar{x}_{n-1} = \frac{n}{n-1} (x_n - \bar{x}_n) \end{equation*}

We replace the rightmost term in \(S_n\) and we finally get

\begin{equation*} S_n = S_{n-1} + (x_n - \bar{x}_{n-1}) (x_n - \bar{x}_n) \end{equation*}

This is the expected formula as displayed in wikipedia.
